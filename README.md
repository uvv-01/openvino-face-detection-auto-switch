# OpenVINO Face Detection with Auto Device Switching 🚀

## 📌 Overview
This project uses OpenVINO to perform real-time face detection with:
- CPU / GPU / AUTO switching
- Live FPS monitoring
- CPU usage tracking
- GUI Control Panel

## ⚡ Features
- Real-time face detection
- Dynamic device switching (CPU/GPU/AUTO)
- Thread-safe inference
- Tkinter GUI dashboard
- Performance monitoring (FPS + CPU usage)

## 🛠️ Tech Stack
- Python
- OpenVINO
- OpenCV
- Tkinter
- psutil

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
